# Portal_Bancario_Usuario
Funcionalidades
 Registro y login con Spring Security (roles: USER, ADMIN).
 Panel con saldo, movimientos y perfil.
 Flujo guiado con Spring Web Flow para:
 Transferencia bancaria
 Recuperación de contraseña
 Actualización de datos
 CRUD de cuentas y usuarios con Hibernate.
 Validaciones en frontend con jQuery + AJAX.
 Mensajes de notificación tipo “Transacción exitosa”.
 
Tecnologías 
 Java 8
 Spring MVC + Spring Security + Spring Web Flow
 JSP + JSTL
 jQuery + AJAX
 Hibernate
 Base de datos: MySQL
